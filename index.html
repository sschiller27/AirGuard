System Overview
How AirGuard
Thinks For You

A plug-in smart module that mounts directly onto any existing purifier — no tools, no rewiring, no modifications. Just instant intelligence.

🔬
Step 01 — Sense
Continuous Air Monitoring
The MQ-135 sensor continuously scans for harmful gases, smoke, VOCs, CO₂, and pollutants. Simultaneously, the DHT22 tracks humidity and temperature in real time.
🧠
Step 02 — Process
Arduino Instant Processing
An Arduino Uno receives both sensor data streams and instantly compares them against safe threshold values, making a decision every few milliseconds.
⚡
Step 03 — Act
Automatic Relay Trigger
When readings cross an unsafe threshold, the Arduino triggers a 5V relay module — automatically switching on your purifier without you lifting a finger.
📺
Step 04 — Display
Live OLED Readouts
A 0.96" OLED screen displays live data from all sensors at all times. Color-coded LEDs give instant visual status: green for clean, red for poor air quality.

Construction
Step-by-Step
Build Plan
01 /
Set Up Arduino IDE
Install the Arduino IDE on your computer and connect the Arduino Uno via USB. Verify the board is recognized under Tools → Board.
02 /
Wire the MQ-135 Sensor
Connect the MQ-135 air quality sensor to the Arduino using analog pin A0. VCC to 5V, GND to GND, AOUT to A0.
03 /
Connect the OLED Display
Wire the OLED via I2C: SDA to pin A4, SCL to pin A5. Power with 3.3V. Install the Adafruit SSD1306 and GFX libraries.
04 /
Wire the LEDs with Resistors
Connect the green LED (with 220Ω resistor) to digital pin 8, and the red LED (with 220Ω resistor) to digital pin 9. GND to common ground.
05 /
Connect the Relay Module
Wire the relay control pin to digital pin 7. Then connect the purifier's power through the relay's COM and NO terminals — have an adult help with AC wiring.
06 /
Upload the Code
Write code that reads sensor data, displays it on the OLED, controls the LEDs, and triggers the relay when readings exceed your threshold. Upload to Arduino.
07 /
Test & Calibrate
Test with clean air first, then simulate poor air near a candle or strong scent. Verify the relay triggers. Adjust the threshold value in code based on your readings.
08 /
Mount in Enclosure
Once everything works, mount all components in a clean white enclosure that attaches to the side of the purifier — no modifications to the purifier itself.
Known Issues
Challenges &
Solutions
⏱ Warm-Up
MQ-135 Needs ~60s Warm-Up
The sensor requires time to stabilize before giving accurate readings. Fix: Add a startup delay and a "warming up" message on the OLED in your code.
⚡ Interference
Relay Electrical Noise
The relay can cause electrical interference with sensor wires. Fix: Route relay wiring away from sensor wires and use a flyback diode if needed.
📡 I2C
OLED Not Displaying
I2C address mismatch is the most common culprit. Fix: Run an I2C scanner sketch — most OLEDs use address 0x3C, but some use 0x3D.
⚠️ Safety
AC Power Wiring
The purifier runs on AC power through the relay. Fix: Always have an adult assist with the AC wiring portion. Never handle live wires alone.
